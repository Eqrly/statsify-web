Commands

Hypixel Guild Stats

Command Description Usage Aliases
-?Head Gets the command to retrieve the head of a player in game. -?head [playername]
-?NameHistory Displays the name history of a player. -?namehistory [playername] namehist, history, hist
-?OptiFine Checks if a player has an OptiFine cape, if so it returns an image of it. -?optifine [playername]
-?Skin Gets the skin of a player. -?skin [playername]
-?Available Checks the availability of a username. -?available [playername]
-?Capes Lists all Minecraft capes. -?capes
-?UUID Gets the UUID of a player. -?uuid [playername]
